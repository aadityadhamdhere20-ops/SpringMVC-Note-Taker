📖 Description

This project is a Note Taker Web Application developed using Spring MVC, Hibernate, and MySQL. It allows users to create, update, delete, and manage notes efficiently.

The application follows a layered MVC architecture and demonstrates real-world web application development with proper validation, database integration, and exception handling.

🚀 Features
➕ Add new notes
✏️ Edit existing notes
❌ Delete notes
📋 View all notes
💾 Store data in MySQL database
⚠️ Exception handling

🧱 Architecture
Controller Layer → Handles user requests
Service Layer → Business logic
DAO Layer → Database operations
Entity Layer → Note model
View Layer (JSP) → UI

🛠️ Technologies Used
Java
Spring MVC
Hibernate (ORM)
MySQL
JSP, HTML, CSS
Maven
Apache Tomcat
